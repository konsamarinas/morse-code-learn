Morse Code Learn


       Morse Code Learn is an amateur and completely free website dedicated to Morse Code. The website covers all the needs one might have regarding code. That is, the visitor has the opportunity to learn the code, solve exercises and practice his skills on this part. Our page is divided into 3 areas. Learning, training and the tool of translation . Learning is a category that teaches you ways to memorize code and how to use/write it. There are 6 lessons where every three lessons there is an exercise. In addition to learning there is a category called training. Here you can practice your coding skills on either easy or hard level. Last but not least is the translation part. Our website offers you an excellent Morse code translator that allows you to translate Morse code to English text and vice versa.
