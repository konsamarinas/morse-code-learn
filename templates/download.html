Application Description


Morse Code Learn publishes a translation app for android mobile devices. This app is an aid to learning Morse code, but also a useful tool for someone who would like to translate code or vice versa. Additionally, when you translate text into code, you can have the translated message played back audibly. This gives you the opportunity to have an audio-visual effect because the code is transmitted by several methods. (see Lesson 5). We hope you like it. Enjoy!



FAQ


Yes, our app is published by Morse Code Learn and is available to everyone for any legal use. Its purpose is purely educational.
The app is only available for android mobile devices (.apk). There are currently no plans to create a desktop app, but there may be in the future.
Our application is completely safe with the guarantee of Morse Code Learn. If you have any doubts about its safety, you can allow Play Protect to run a check after installing the app to make sure.
If your mobile gives you a warning when you open the file and does not let you install it, it means that you have not turned on the installation of applications from unknown sources outside the Play store. To solve this, open settings-privacy/security-unknown sources.
If you get a notification from Play Protect when you press install, press the arrow below and then "install anyway". If you want to make sure that the app is completely safe, play protect will show you a notification at the end that asks you if you want to scan it.
To see if there is a new update, all you have to do is come to this page and check for updates.


If you have any questions about installing the app, or want to report any bugs you've gotten, don't hesitate to contact us. Pleased to help you!
